# Aspect Lexicon

Seed keywords used to tag reviews with the service aspects they mention. Each aspect below lists
example English seed keywords. Arabic seed equivalents are maintained alongside each aspect and
are expanded together with the English seeds during annotation and iteration.

| Aspect | Example English seed keywords |
|---|---|
| Login / authentication | login, sign in, log in, authentication, password, OTP, biometric |
| Payment | payment, pay, fee, transaction, card, refund, checkout |
| Usability | easy, intuitive, confusing, interface, navigation, layout, user-friendly |
| Performance | slow, fast, lag, crash, freeze, loading, responsive |
| Updates | update, version, upgrade, patch, release, changelog |
| Support | support, help, customer service, contact, response, ticket |
| Verification | verify, verification, identity, document, approval, confirm |
| Notifications | notification, alert, reminder, push, message, badge |
| Language | language, Arabic, English, translation, localisation, RTL |
| Registration | register, registration, sign up, account, onboarding, create account |

**Note:** Arabic seed equivalents for every aspect are maintained alongside the English seeds
listed above and are curated together so that bilingual coverage stays aligned.
